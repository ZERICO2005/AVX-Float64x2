Build Instructions for Windows 10 MSYS2 MinGW64:
Configure the CMakeLists.txt if needed. From this directory, run `mkdir build`, `cd build`, `cmake -G Ninja ..`. Then, run `ninja` to compile, and `bin/Dekker-Float-Test` to run the program.
